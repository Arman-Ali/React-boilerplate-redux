<h1>Documentation</h1>
<h2>Quickstart</h2>
<ol>
  <li>First, let's clone React-boilerplate-redux repository then you need to install the dependencies:</li>
  <code>npm install</code>
  <li>Now you can run the project:</li>
  <code>npm start</code>
  <li>Open localhost:3000 to see it in action.</li>
</ol>
